# 풍선 경주
사다리타기처럼 한 사람의 술래를 뽑을 때 사용할 수 있습니다.
가장 빨리 결승선을 넘은 풍선의 주인을 뽑는 '승자 가려내기'와, 가장 느리게 떠올라 마지막에 남은 풍선의 주인을 뽑는 '꼴찌 가려내기' 모드를 선택할 수 있습니다.
